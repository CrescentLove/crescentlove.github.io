linux&server

整理一下近期常用的一些shell命令，更多的shell变量函数移步菜鸟教程查手册

所有命令均可以使用以下方式查看手册和帮助,所以我更多的只是列一些常用方法

cd & ls & pwd

..等目录结构见附录

mkdir

权限777等文件权限见附录

cp & rm & mv

tar

附

特殊字符

& | < > * ? ' " ; [ - ] # \

正则的小部分知识（查表完事记两个常用即可）

*用来匹配任意字符串（可为空）

?用来匹配单个字符（不可为空），且? *均不能匹配/

[ - ]用来匹配方括号里的字符，hyphen用来表示字符范围（仅在方括号内生效）

目录结构

..向上一级根目录

.当前目录的绝对路径

/根目录

~用户主目录（home）

尽量使用以./xxx或者/xxx开头的绝对路径，避免相对路径优先在预设的搜索路径里查找文件

文件权限

按照文件主，组用户和其他用户的分类

user group others
rwx rwx rwx

rwx按二进制赋权再按8|10进制转化为权限码

比如rw-r--r--按二进制赋权421再转化为8进制即为644

比如rwxrwxrwx按二进制赋权421再转化为8进制即为777

命令缩写

cat catenate
ls list files
cd change directory
mkdir make directory
cp copy file
rm remove file
mv move file
chmod change mode
