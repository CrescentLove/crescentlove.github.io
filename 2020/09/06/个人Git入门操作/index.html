Git入门

上传本地库例子

.1 准备工作

  • 安装git
  • 注册Gitee账号（GitHub同理）

.2 本地库操作

  • 在你的项目文件夹里右键打开git bash，并新建仓库，此时这才算是一个本地库而不只是一个本地文件夹。

  • 接下来需要将项目文件夹中的文件都添加到本地库中，上一步虽然在文件夹里新建了.git文件，但是他们貌合神离需要再次整合。

  • 接下来需要将文件提交到本地库中

到这里本地库操作已经完成了。

.3 Push！

要将本地库提交到gitee上，首先得给它在gitee上新建一个容身的仓库或是直接提交到已有的项目仓库里。

  • 新建项目仓库，并获取仓库地址

  • 将本地仓库与远程仓库进行关联

  • 最后，将本地仓库里的文件push到远程仓库，这样你的轮子也能在网上滚动了。

注意，上传时经常会出现这样的问题：

按照提示，需要首先将远程仓库的更新pull到本地。

版本控制

git log –pretty=oneline 查看提交历史
git reflog 查看历史命令（获得commit id）
git reset –hard xxx 回退版本到xxx提交处
